QuantelScan Installation instructions:

1.  Run the vc_redist.x86 file
2.  Check the box to agree to license terms and condtions.
3.  Click Install.
4.  Click Close when the installation is complete.
5.  QuantelScan can now be copied from the USB drive to any location on the computer.
6.  Run Quantelscan (Refer to the Viron Manual for instructions to use QuantelScan).
